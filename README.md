# 🎮 Smart Learning Educational Game

A gamified Android learning application designed to improve student engagement and learning effectiveness through interactive educational gameplay. The system transforms traditional learning activities into RPG-style quests, allowing students to progress through educational challenges while tracking their academic achievements.

---

## ✨ Key Features

### 1. Role-Based Learning System

**Student Portal**

* Complete educational quests and assignments.
* Participate in PvE (Player versus Environment) learning battles.
* Earn EXP, level up, and unlock achievements.
* View personal progress, performance statistics, and leaderboard rankings.
* Receive real-time feedback after answering questions.

**Teacher Portal**

* Create and manage educational content.
* Assign learning quests and activities.
* Monitor student performance and progress.
* Review student achievements and mastery levels.
* Manage classroom learning activities efficiently.

---

### 2. Gamified Learning Engine

The application integrates RPG-inspired mechanics to increase motivation and engagement through:

* **Experience Points (EXP)** – Students gain EXP by completing quizzes and learning tasks.
* **Level Progression** – Users level up as they accumulate knowledge and complete quests.
* **Health Points (HP)** – Represents player status during learning battles.
* **PvE Battle System** – Students answer academic questions to defeat enemies and complete missions.
* **Achievement Tracking** – Rewards students for consistent participation and learning progress.

---

### 3. Interactive Tutorial & Onboarding

The system provides a dynamic onboarding experience that introduces users to the world of Numeria through:

* Story-based tutorial videos.
* Interactive learning guides.
* Role-specific tutorials for students and teachers.
* Visual demonstrations of game mechanics and learning objectives.

---

### 4. Real-Time Progress Tracking

Integrated progress monitoring allows users to:

* Track EXP and level growth.
* Monitor completed quests and learning activities.
* View mastery scores and learning achievements.
* Access leaderboard rankings for motivation and competition.

---

### 5. Learning Content Management

Teachers can efficiently manage educational materials through:

* Question creation and modification.
* Quest assignment and monitoring.
* Student performance evaluation.
* Classroom activity management.

## ⚙️ Setup

1. Clone the repository.
2. Open the project using Android Studio.
3. Connect your Firebase project.
4. Add the `google-services.json` file into the `/app` directory.
5. Enable Firebase Authentication and Cloud Firestore.
6. Build and run the application on an Android device or emulator.
